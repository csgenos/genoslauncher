GenosLauncher

A fast, clean Minecraft launcher built for players who want more — modpack support, shader management, and seamless Microsoft sign-in.

⬇  Download for Windows View on GitHub

Windows 10 / 11  ·  No admin required  ·  Free forever

Features

Everything you need, nothing you don't

🔐

Microsoft Sign-In

Log in with your real Microsoft account — no Azure registration, no browser redirects, no setup. Just enter a code and you're in.

📦

Modpack Support

Import any .mrpack file. GenosLauncher automatically installs the right Fabric or Quilt version and all mods — no manual steps.

✨

Iris & Sodium

Install Iris Shaders and Sodium with a single click, verified against official Modrinth hashes. Then drop in any shader pack you want.

⚡

Smart JVM Tuning

Choose from Performance, Balanced, or Custom JVM presets. GenosLauncher detects your system RAM and recommends the right allocation.

🗂️

Instance Management

Run multiple separate Minecraft instances — different versions, different mod sets — each with its own isolated directory and settings.

🎨

Clean Design

A polished dark UI with a first-run setup wizard. Feels like Lunar Client — not a hobby project thrown together on a weekend.

Getting started

Up and running in three steps

1

Download the installer

Click the button above to download the Windows installer. No admin rights required — it installs to your user folder, just like Discord or Cursor.

2

Run the setup wizard

On first launch a quick wizard walks you through signing in with Microsoft (or adding an offline account) and setting your RAM. Takes under a minute.

3

Play

Pick a version, hit Play. Java is detected automatically. Install modpacks and shaders from the built-in tabs whenever you want.

Ready to play?

Free download. No account needed to try it.
